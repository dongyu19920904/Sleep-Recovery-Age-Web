# AI生物年龄工具项目雷达

## 今日优先项目

### [SaraYCano 全脑年龄模型项目](https://github.com/SaraYCano/Antomical-modelo-of-the-whole-brain-associated-with-age)

这个项目用 BOLD 信号和 DTI 结构连接性观察全脑模型与年龄关系，是脑龄和数字生物标志物方向的早期线索。它不成熟，但适合做「脑龄工具雷达」的第一张卡片。

- 证据来源：GitHub 仓库，线上项目雷达 2026-06-22 收录
- 可信度：中低
- 它解决什么问题：尝试用神经影像数据识别大脑老化相关指标
- 可做成什么：内容拆解 / 脑龄术语库 / README 试读 / 长期观察页
- 售后或合规风险：中，不能用于诊断或健康建议
- 今天最小动作：读 README，整理 BOLD、DTI、熵产生三个词条

## 可二次开发方向

- **脑龄术语库**：把 brain age、BOLD、DTI、connectome、digital biomarker 做成解释卡片。
- **README 试跑笔记**：不急着跑模型，先记录环境、数据来源和可复现性。
- **工具导航卡片**：每个脑龄项目保留 star、语言、数据类型、风险说明。
- **可穿戴可信度对比页**：从医疗设备校准话题延伸，整理健康设备的公开标准。

## 值得观察

- `lucascamillomd/pyaging`：成熟的 aging clocks 工具，适合后续做教程。
